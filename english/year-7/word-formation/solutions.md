# Solutions - Word Formation Practice (Detailed)

## Part 1: Suffixes and Prefixes
1. **explanation** (Root: explain. Verb -> Noun transition using suffix -ation).
2. **advisable** (Root: advise. Verb -> Adjective. Note: drop the 'e' before adding -able).
3. **commitment** (Root: commit. Verb -> Noun. Doubling the final 't' is crucial).
4. **appearance** (Root: appear. Verb -> Noun using suffix -ance).
5. **kindness** (Root: kind. Adjective -> Noun using suffix -ness).
6. **growth** (Root: grow. Verb -> Noun. Irregular change from 'grow' to 'growth').
7. **solution** (Root: solve. Verb -> Noun using suffix -tion. Note the spelling change from 'v' to 'u').
8. **behaviour / behavior** (Root: behave. Verb -> Noun. Both British and American spellings are correct).
9. **variation / variety** (Root: vary. Verb -> Noun. 'Variation' refers to a change; 'variety' refers to a range of types).
10. **strength** (Root: strong. Adjective -> Noun. Irregular vowel change).

## Part 2: Advanced Word Formation
1. **Unemployment** (Root: employ. Suffix -ment for noun; Prefix un- to indicate the lack of jobs as an issue).
2. **threat** (Root: threaten. Verb -> Noun. 'Threaten' is the action; 'threat' is the noun form).
3. **memorable** (Root: memory. Noun -> Adjective. Something worth remembering).
4. **description** (Root: describe. Verb -> Noun. Note the spelling change from 'b' to 'p' before adding -tion).
5. **Curiosity** (Root: curious. Adjective -> Noun using suffix -ity).
6. **maintenance** (Root: maintain. Verb -> Noun. Note the spelling change from -ain to -en before adding -ance).
7. **remarkable** (Root: remark. Verb -> Adjective. Something worthy of notice).
8. **Confluence** (Root: confluence - note: provided as root but testing prefix/suffix understanding. Usually refers to the flowing together of things).
9. **accuracy** (Root: accurate. Adjective -> Noun using suffix -acy).
10. **generosity** (Root: generous. Adjective -> Noun using suffix -osity).
